Install Django
i 
                        python --version || python3 --version
i
This command is to check the current version of Python installed on your system.
Why: Being a Python Web Framework, Django requires Python. By running above command, you can verify that Python is installed on your system. and you should see something like: 
                            
                                python 3.x.y
Getting error? i.e. Python is not installed on your system. Don't worry, Get the latest version of Python from its official site https://www.python.org/downloads/ or your operating system package manager. 
                        pip install django
i
This command will install Django on your system, if not installed.
To verfiy that Django is installed on your system run Python Prompt by running python in your shell and then at the Python Prompt try to import Django: 
                            
                                $ python
                                Python 3.7.6 (default, Jan  8 2020, 19:59:22) 
                                [GCC 7.3.0] on linux
                                Type "help", "copyright", "credits" or "license" for more information.
                                >>> import django
                                >>> print(django.get_version())
                                3.0.5
                                >>>
You may have another version of Django installed
