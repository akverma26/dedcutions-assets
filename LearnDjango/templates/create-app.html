Create App
i 
                        
                            python manage.py startapp [app_name]
i
This command will create an App named [app_name] with all configuration files, that is neccessary for rendering a web page.
Till now we have just setup our Django environment and only can see "Congratulations" page by Django. To show our custom pages first we have to create an App. For sake of understanding, you can consider an App as a single page of a website or a single service provided by website. But yes you can create all pages and services of a website from one app, but separating all these things into different Apps make your project more convenient. You can create as many app as you want.
You can create an app by running above command that will automatically create a folder containing some configuration files, this folder will look like this: 
                                
                                    |- [app_name]/
                                        |- admin.py
                                        |- apps.py
                                        |- models.py
                                        |- tests.py
                                        |- views.py
views.py is the file that is responsible for rendering your custom page or view. How to do that read the section Show Custom View
